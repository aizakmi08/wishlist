# Wishlist Mobile App

A React Native and Expo prototype for managing wishlists, social gifting, events, and notifications from a mobile-first interface.

## Product Scope

- Home dashboard for upcoming events and recent activity.
- Event management for birthdays, Secret Santa, and custom gifting occasions.
- Wishlist screens for creating and reviewing gift ideas.
- Profile and settings screens for user-facing account state.
- Notification center for event and wishlist updates.
- Dark mobile interface with bottom tab navigation and Lucide icons.

## Tech Stack

- Expo SDK 54
- React Native 0.81
- React 19
- React Navigation
- Expo Blur
- Lucide React Native
- React Native SVG
- React Native Safe Area Context

## Run Locally

```bash
npm install
npm start
```

Then open the app in Expo Go, an iOS simulator, Android emulator, or web:

```bash
npm run ios
npm run android
npm run web
```

## Status

This is a mobile product prototype focused on navigation, screen composition, and polished UI states. It is useful as a front-end example rather than a complete production backend.
